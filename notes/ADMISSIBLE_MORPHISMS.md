# Admissible BEpTy Morphisms

## Definition
\[
\Phi:G\to H
\text{ is an admissible BEpTy morphism}
\]
\[
\Longleftrightarrow
\Phi\text{ induces }
\Phi_\ast:Z_1(G)\to Z_1(H),
\qquad
\Phi_\ast(X_R(G))\subseteq X_R(H),
\]
\[
\text{and preserves minimal support size on each nonzero class of }Z_1(G)/X_R(G).
\]

## Invariance lemma
\[
\Phi:G\to H\text{ admissible}
\Longrightarrow
\beta^{\mathrm{prof}}_R(G)=\beta^{\mathrm{prof}}_R(H)
\]
\[
\text{whenever }\Phi_\ast:Z_1(G)/X_R(G)\xrightarrow{\sim} Z_1(H)/X_R(H).
\]

## Proof skeleton
\[
\Phi_\ast(X_R(G))\subseteq X_R(H)
\Longrightarrow
\Phi_\ast\text{ descends to }
\overline{\Phi}_\ast:Z_1(G)/X_R(G)\to Z_1(H)/X_R(H).
\]
\[
\overline{\Phi}_\ast\text{ is an isomorphism}
\Longrightarrow
\dim Z_1(G)-\dim X_R(G)=\dim Z_1(H)-\dim X_R(H).
\]
\[
\text{Minimal support size is preserved on nonzero quotient classes}
\Longrightarrow
\sigma_R(G)=\sigma_R(H).
\]
\[
\therefore\quad
\beta^{\mathrm{prof}}_R(G)=\beta^{\mathrm{prof}}_R(H).
\]
