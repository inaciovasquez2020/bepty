# Sheaf Comparison Class

## Fixed comparison class
For the current BEpTy program, define the provisional comparison class by
\[
\mathcal I_{\mathrm{sheaf}}(G)
:=
\bigl(
\dim H_0(G;\mathbb F_2),
\dim H_1(G;\mathbb F_2),
\sigma_{\mathrm{cell}}(G)
\bigr),
\]
where \(\sigma_{\mathrm{cell}}(G)\) denotes the isomorphism type of the constant-\(\mathbb F_2\) cellular sheaf on \(G\).

## Separation target
Find \(G,H,R\) such that
\[
\mathcal I_{\mathrm{sheaf}}(G)=\mathcal I_{\mathrm{sheaf}}(H)
\quad\text{and}\quad
\beta^{\mathrm{prof}}_R(G)\neq\beta^{\mathrm{prof}}_R(H).
\]

## Current status
Known:
\[
\beta^{\mathrm{prof}}_1(\Theta_{3,3,3})\neq\beta^{\mathrm{prof}}_1(D_{6,6}).
\]

Unproved:
\[
\mathcal I_{\mathrm{sheaf}}(\Theta_{3,3,3})=\mathcal I_{\mathrm{sheaf}}(D_{6,6}).
\]
