# Sheaf Nonrecoverability Target

## Comparison class
Fix a sheaf-theoretic comparison class
\[
\mathcal I_{\mathrm{sheaf}}(G).
\]

## Target statement
\[
\exists G,H,\exists R\ge 0,\qquad
\mathcal I_{\mathrm{sheaf}}(G)=\mathcal I_{\mathrm{sheaf}}(H)
\quad\land\quad
\beta^{\mathrm{prof}}_R(G)\neq \beta^{\mathrm{prof}}_R(H).
\]

## Current established result
\[
\beta^{\mathrm{prof}}_1(\Theta_{3,3,3})=(2,\{\!\{6,6,6\}\!\}),
\qquad
\beta^{\mathrm{prof}}_1(D_{6,6})=(2,\{\!\{6,6,12\}\!\}),
\]
\[
\beta^{\mathrm{prof}}_1(\Theta_{3,3,3})\neq \beta^{\mathrm{prof}}_1(D_{6,6}).
\]

Open:
