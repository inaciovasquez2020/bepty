# C4 vs Tree Demo

## Cycle graph case

Let \(G=C_4\) and \(R=1\).

\[
\dim_{\mathbf F_2} Z_1(C_4)=1.
\]

Each radius-1 ball is a path, hence acyclic, so

\[
Z_1(B_1(v))=0
\quad\text{for all }v.
\]

Therefore

\[
X_1(C_4)=Z_1(C_4),
\qquad
\beta^{\dim}_1(C_4)=1,
\qquad
\mathrm{LocalExhaustion}_1(C_4)=1.
\]

## Tree case

Let \(G=T\) be any tree.

\[
Z_1(T)=0.
\]

Therefore

\[
X_R(T)=0,
\qquad
\beta^{\dim}_R(T)=0,
\qquad
\mathrm{LocalExhaustion}_R(T)=0
\]

for every \(R\).
