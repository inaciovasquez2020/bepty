# BEpTy Quickstart

This is the shortest path from clone to a first successful local verification pass.

## Requirements

- `git`
- `bash`
- `python3`

## 1. Clone

```bash
git clone https://github.com/inaciovasquez2020/bepty.git
cd bepty
```

## 2. Check tools

```bash
python3 --version
git --version
```

## 3. Install test tooling

```bash
python3 -m pip install pytest
```

## 4. Run tests

```bash
python3 -m pytest -q
```
