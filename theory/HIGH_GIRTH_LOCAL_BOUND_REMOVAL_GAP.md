# HIGH_GIRTH_LOCAL_BOUND_REMOVAL_GAP

Status: OPEN

## Statement

Construct an admissible canonical valuation scheme V_r on rooted radius-r profile data such that

P_r(G) != P_r(H) => V_r(G) != V_r(H)

for all finite bounded-degree graphs G,H,

without dependence on an a priori coordinate bound M.

## Role

This is the exact gap between
HIGH_GIRTH_LOCAL_WEIGHTING_BASE_B_CONDITIONAL
and
HIGH_GIRTH_LOCAL_VALUATION_LEMMA.

## Reduction

bound_removal => HIGH_GIRTH_LOCAL_VALUATION_LEMMA
