# HIGH_GIRTH_LOCAL_VALUATION_LEMMA

Status: OPEN

## Statement

Fix r >= 1.

Let P_r(G), P_r(H) be rooted radius-r profile count vectors for finite bounded-degree graphs G,H.

If

P_r(G) != P_r(H),

then there exists an admissible BEpTy valuation V_r depending only on rooted radius-r profile data such that

V_r(G) != V_r(H).

## Role

This is the single weakest missing lemma for
HIGH_GIRTH_LOCAL_EXHAUSTION_THEOREM.

## Reduction

profile_vector_difference => valuation_difference
