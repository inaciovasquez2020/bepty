# HIGH_GIRTH_LOCAL_VALUATION_CONSTRUCTION

Status: CONDITIONAL

## Candidate Construction

Fix r >= 1.

Let P_r(G) = (p_1(G), ..., p_m(G)) be the rooted radius-r profile count vector
with respect to a fixed canonical ordering of rooted radius-r types.

Define the admissible BEpTy local valuation

V_r(G) = \sum_{i=1}^{m} i * p_i(G).

## Conditional Claim

If P_r(G) != P_r(H) and the first differing coordinate is order-detecting
under the canonical indexing, then

V_r(G) != V_r(H).
