## Spectral-gap extraction statement

A spectral gap is the existence of \(m>0\) such that
\[
\operatorname{Spec}(H)\cap(0,m)=\varnothing.
\]

Equivalently,
\[
\boxed{
\forall \psi\in\Omega^\perp,\qquad
\langle \psi,e^{-tH}\psi\rangle \le e^{-mt}\|\psi\|^2
\quad\forall t\ge 0.
}
\]

## Conditional extraction lemma

Assume there exists a canonical reflected observable \(F\) from the frozen base family such that its reconstructed vector
\[
\psi_F := [F] \in \mathcal H
\]
satisfies
\[
\psi_F \perp \Omega
\]
and its Euclidean two-point function obeys the exponential bound
\[
\boxed{
\langle \Omega,\Theta F\, \tau_t(F)\Omega\rangle \le C e^{-mt}
\quad\forall t\ge 0
}
\]
for some \(C>0\), \(m>0\).

Then
\[
\boxed{
\langle \psi_F,e^{-tH}\psi_F\rangle \le C e^{-mt}
\quad\forall t\ge 0,
}
\]
and hence \(H\) has spectral support bounded below by \(m\) on the cyclic subspace generated by \(\psi_F\).
